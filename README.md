# React

Learning React

[React Best Practices](https://github.com/airbnb/javascript/tree/master/react)

1. Create an app called "my-react-app"

```
npm create vite@latest my-react-app
```

2. Change directory

```
cd my-react-app
```

3. Install dependencies

```
npm install
```

4. Run app

```
npm run dev
```
